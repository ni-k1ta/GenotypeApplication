# GenotypeApplication

A graphical application for automating genotyping data processing and population structure analysis. The application integrates the execution and result processing of the following external programs: **Structure**, **Structure Harvester**, **CLUMPP**, and **Distruct**.

## Requirements

- Windows
- .NET 8.0

## Running the Application

The application requires the executable files of external programs located in the `External programs` folder. Make sure all files from this folder are present next to the application executable before launching.

### External Programs

All listed files must be placed in the `External programs` folder alongside the application executable.

| File | Description |
|---|---|
| `structure.exe` | A program for clustering individuals based on genotypic data using Bayesian methods. Determines the optimal number of populations (K) and assigns individuals to clusters. |
| `structureHarvester.exe` | A tool for processing and analyzing Structure output data. Calculates the ΔK metric (using the Evanno method) to identify the most likely number of populations. |
| `CLUMPP.exe` | A program for aligning and averaging results from multiple independent Structure runs, resolving label-switching ambiguity across runs. |
| `distructWindows1.1.exe` | A program for visualizing clustering results as bar plots showing the membership of individuals to populations. |
| `gswin64c.exe` | The Ghostscript interpreter — used for rendering and converting PostScript/PDF files generated by Distruct. |
| `gsdll64.dll` | The Ghostscript library required for `gswin64c.exe` to function. |
